# Judge [CREATIVE] 创意加分区

> ⚡ 强模型加分区 — 弱模型可完整跳过本节，创意评分默认记 0 分。不影响 Verdict。

## 创意评分（0-20 分，独立于结构评分）

在结构核查完成后，对以下 4 个维度各打 0-5 分：

| 维度 | 0分（不合格） | 3分（合格） | 5分（优秀） |
|---|---|---|---|
| **隐喻一致性** | style_prompt 风格与 DNA 隐喻无关联 | 有对应但表浅 | 设计语言直接体现 DNA 核心隐喻 |
| **情绪传达** | 文案/视觉无法让目标用户感受到情绪目标 | 可感受到但不强烈 | 打开即能感受到 DNA 情绪目标 |
| **差异化** | 风格套用通用模板，无品牌辨识度 | 有少量特色细节 | Bold Factor 执行到位，一眼可识别 |
| **细节品质** | 存在明显视觉/交互粗糙点 | 基本完整 | 微交互、间距、层级无明显短板 |

评分输出格式（JSON）：
```json
{
  "creative_score": {
    "total": 16,
    "max": 20,
    "label": "✨ 高质量",
    "dimensions": [
      {"name": "隐喻一致性", "score": 4, "note": "..."},
      {"name": "情绪传达", "score": 4, "note": "..."},
      {"name": "差异化", "score": 4, "note": "..."},
      {"name": "细节品质", "score": 4, "note": "..."}
    ]
  }
}
```

标记阈值：✨ 高质量（≥15）/ ✓ 合格（10-14）/ ⚠️ 建议润色（<10）
