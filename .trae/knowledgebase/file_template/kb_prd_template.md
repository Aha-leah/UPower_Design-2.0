## 六、人类决策点（Stop Conditions / 必须追问）

当出现以下任意情况，禁止猜测执行，必须向用户追问并等待确认：
1.  **情绪目标缺失** 或 与隐喻冲突
2.  **主终端/首要动作不明确**
3.  **成功标准无法验收**（没有口径/没有目标值）
4.  **范围冲突**（In/Out Scope 含糊或互相打架）
5.  **关键约束缺失**（必须遵循 tokens/规范但未提供路径或规则）
